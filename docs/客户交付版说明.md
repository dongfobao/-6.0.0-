# 客户交付版说明

这套“客户交付版”解决的是客户现场使用门槛问题。

## 交付方式

最终推荐交付的是 `release\YLDQ数据分析系统` 目录。

客户拿到后只需要：

1. 把实际数据放进同目录 `实时数据`
2. 双击 `YLDQ数据分析.exe`
3. 自动生成 `output\分析报告.html`
4. 浏览器自动打开报告

## 技术路线

客户交付版不再依赖本地 HTTP 服务来查看结果，而是：

1. 直接解析 `实时数据`
2. 生成一个独立的静态 HTML 报告
3. 同时输出 `txt` 和 `json`
4. 自动调用浏览器打开 HTML

这样做的好处：

1. 客户电脑不需要 Python 环境
2. 不需要开放端口
3. 不依赖本地服务常驻
4. 更适合售后和客户现场传递
