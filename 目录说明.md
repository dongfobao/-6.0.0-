# 目录说明

本目录按用途整理为以下几类：

- `app/`：分析系统主程序源码，包含后端 Python 模块和前端 `web/` 页面资源。
- `scripts/`：启动、打包、生成客户报告等辅助脚本。
- `packaging/`：PyInstaller 打包配置和版本资源。
- `tests/`：离线单元测试。
- `docs/`：设计文档、客户说明和历史规格文档。
- `release/`：已打包的可执行程序和压缩包，交付或拷贝使用。
- `实时采集会话/`：现场采集产生的会话数据。

根目录保留常用入口脚本：

- `启动分析系统.bat`
- `打包分析系统EXE.bat`
- `打包客户交付版EXE.bat`
- `生成客户版报告.bat`

`build/` 和 `dist/` 是 PyInstaller 中间产物，可随时重新生成，不需要长期保留。
