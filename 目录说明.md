# 目录说明

- `app/`：YLDQ 6.0 远程监控后端、Modbus V7 协议核心及 Web 前端。
- `scripts/`：源码启动和 PyInstaller 打包脚本。
- `packaging/`：远程监控程序打包配置。
- `tests/`：协议、采集、配置事务、记录和界面投影测试。
- `docs/design/`：上位机架构设计。
- `实时采集会话/`：监控运行时生成的会话数据（不提交 Git）。
- `release/`：重新打包生成的交付目录（不提交 Git）。

常用入口：

- `启动源码调试.bat`：运行源码版并打开浏览器。
- `启动分析系统.bat`：优先启动新打包版本，没有时启动源码版。
- `scripts/打包分析系统EXE.bat`：生成 YLDQ 6.0 远程监控程序。
